구현
===
> 머릿속에 았는 알고리즘을 소스코드로 바꾸는 과정으로   
> 모든 범위의 코딩테스트 문제 유형을 포함하는 개념이다.

- 완전 탐색, 시뮬레이션 유형을 모두 '구현' 유형으로 묶어 다룬다.
  - 완전 탐색 : 모든 경우의 수를 주저 없이 다 계산하는 해결 방법   
  - 시뮬레이션 : 문제에서 제시한 알고리즘을 한 단계씩 차례대로 직접 수행해야 하는 문제 유형
