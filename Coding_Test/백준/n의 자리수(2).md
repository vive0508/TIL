## BAEKJOON ONLINE JUDGE
### 단계별로 풀어보기
#### 03. 반복문 ([1110번](https://www.acmicpc.net/problem/1110))

```python
n = int(input())
num = n
cnt = 0

while True:
    # 십의 자리 숫자
    a = num // 10
    # 일의 자리 숫자
    b = num % 10
    c = (a+b) % 10
    num = (b*10) + c
    
    cnt = cnt + 1
    if(num == n):
        break

print(cnt)
```
