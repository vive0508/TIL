#### 02. 조건문 ([2884번](https://www.acmicpc.net/problem/2884))

> 원래 설정되어 있는 알람을 45분 앞서는 시간으로 바꾸시오   
```python
hour, min = map(int,input().split())
if min >= 45:
    print(hour, min-45)
elif hour>0 and min < 45:
    print(hour-1, min+15)
else:
    print(23, min+15)
```

1시간을 빼준 뒤 15분을 더하면, 45분을 앞선것과 같음
